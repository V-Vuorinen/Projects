# - Lets start by plotting a barplot from sns and try other plotting types
# - Here we can see that cars had a significantly high average speed compared to other //
# - specifications. The distribution inside the cars specification was also was very low, //
# - when for example the values within birds had a more vibrant distribution.

plt.figure(figsize=(12, 6))
sns.barplot(x='Specification', y='Top Speed', data=df)
plt.title('Average Top Speed by Specification')
plt.show()
In[66]:
# - Lets try to understand better the distribution of values per specification type //
# - we will use a boxplot for this. We better understand the significanse of the distribution //
# - within birds. We also see that the Tennis ball hits, cars and fish had low variance //
# - within the categories. 

plt.figure(figsize=(12, 6))
sns.boxplot(x='Specification', y='Top Speed', data=df)
plt.title('Top Speed Distribution by Specification')
plt.show()
